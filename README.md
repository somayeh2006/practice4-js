Array Sorting and Joining
A simple JavaScript example that sorts a list of colors and joins them into a single string.

Sorts the array alphabetically using sort()
Joins the array items into a string using join()
Logs the results to the console
Purpose
Practice array sorting
Learn how sort() and join() work together
Understand basic array manipulation
